# Reciprocity

> A modern platform for exchanging services without money.

## About

Reciprocity is a web platform that connects people interested in exchanging services instead of paying for them.

The goal is to encourage collaboration between people by allowing users to offer their skills and request services from others through a simple, secure and intuitive experience.

This project is being developed from scratch as a portfolio project while studying modern web development.
