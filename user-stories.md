# User Stories

## Account Creation

As a visitor,
I want to create an account,
so that I can access the platform and publish my services.

---

## User Authentication

As a registered user,
I want to log in,
so that I can access my profile and manage my activities.

---

## Service Registration

As a user,
I want to register my skills and services,
so that other users can discover what I can offer.

---

## Service Discovery

As a user,
I want to browse available services,
so that I can find people who offer skills that match my needs.

---

## Exchange Request

As a user,
I want to request a service exchange,
so that I can negotiate a collaboration with another person.

---

## Request Management

As a user,
I want to view my exchange requests,
so that I can monitor pending, accepted, or rejected collaborations.

---

## Profile Management

As a user,
I want to manage my profile information,
so that other people can better understand my skills and interests.
